Learning intentions.

Key results

For $x^2+bx$, add and subtract $\left(\tfrac b2\right)^2$ to make $\left(x+\tfrac b2\right)^2$. Then $(x+p)^2=q$ gives $x=-p\pm\sqrt q$. If $q<0$ there is no real solution.

Warm-up — what number completes the square?

1. State the number to add to $x^2+bx$ to form a perfect square.
a $x^2+6x$
b $x^2-10x$
c $x^2+14x$
d $x^2-3x$

Part A — Completed-square form

2. Write each in the form $(x+p)^2+q$.
a $x^2-2x-4$
b $x^2+8x+3$

Part B — Solving (whole-number $b$)

3. Solve by completing the square. Leave surds exact.
a $x^2-2x-4=0$
b $x^2+8x+3=0$
c $x^2-6x+4=0$
d $x^2+2x-1=0$
