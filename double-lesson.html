110-minute revision sequence

Double revision lesson

A two-period lesson for diagnosing skills, rebuilding methods, and finishing with focused mixed practice. Students can work directly from this page.

2 x 55 minutes Lesson 1: diagnose and rebuild Lesson 2: apply and consolidate Open student task pack

Student task pack

What each student needs

Use a notebook or loose-leaf paper for working. Each answer should include the rule used, the key algebra line, and a clearly labelled final answer.

Teacher flow

Lesson 1: diagnose and rebuild

Start broad, then let students target the topic areas that need the most immediate repair.

0-8 min

Readiness check

  1. State the domain of f(x)=x+3.
  2. Solve x2-5x+6=0.
  3. Find the period of y=3sin(2x).
  4. Find Pr(A∩B) when Pr(A)=0.6 and Pr(B|A)=0.25.
  5. Find the midpoint of A(-4,7) and B(2,-1).
  6. State whether h(x)=-2x-1+5 has a maximum or minimum value, and give that value.
8-18 min

Traffic-light marking

  • Green: correct method and correct answer.
  • Amber: correct idea but incomplete notation or arithmetic error.
  • Red: unsure how to start.
  • Choose two red or amber topics for the station block.
18-38 min

Skill stations

Choose two stations based on your traffic-light result. Complete all questions in the chosen stations before moving to extension.

Functions For f(x)=10-2x, state the domain and range, then write full function notation.
Trig Solve 2cos(x)+1=0 for 0≤x≤2π, then state the reference angle.
Quadratics A quadratic has turning point (-1,6) and passes through (1,-2). Find its rule.
Transformations Describe the transformations from y=x to y=3x+4-2.
Probability A two-stage event has Pr(A)=0.45, Pr(B|A)=0.8. Find Pr(A∩B).
Coordinate geometry For P(-3,4) and Q(5,0), find distance, midpoint, gradient, and the line equation.
38-50 min

Error hunt

Inverse function checkpoint

A common error is to swap x and y but forget to swap the domain and range. Correct the rule, then state the new domain.

Trigonometric period checkpoint

A common error is to multiply the period by the coefficient inside the bracket. For sine and cosine, use 2π|n|.

Domain checkpoint

For square-root functions, set the expression under the root greater than or equal to zero before writing interval notation.

50-55 min

Reset checkpoint

Write one sentence: "The topic I need to stabilise before mixed practice is..." Then write the first action that would improve it.

Second period

Lesson 2: apply and consolidate

Move from targeted repair into timed, mixed, clearly communicated mathematical work.

0-5 min

Re-entry warm-up

Choose the matching quick question from your checkpoint topic.

  1. Functions: state the domain of x-6.
  2. Trig: find the period of y=tan(4x).
  3. Quadratics: calculate the discriminant of 2x2-3x+5.
5-25 min

Timed mixed practice

  1. For p(x)=2x+8-3, state the maximal domain and range.
  2. Solve x2-7x+10=0.
  3. For x2+kx+9=0, find the values of k that give one repeated real solution.
  4. Sketch y=-2sin(x)+1 for 0≤x≤2π.
  5. A repeated independent trial has success probability 0.2 and is performed three times. Find the probability of at least one success.
  6. Find the equation of the line through (-2,8) and (4,2).
  7. Find the inverse of q:[0,∞)→R, q(x)=x2+5.
  8. Create a two-piece function with one linear rule and one quadratic rule over adjacent non-overlapping domains.
25-38 min

Pair explanation

  • Choose one response from timed practice.
  • Explain the first line, the key method step, and the final answer.
  • Partner checks notation, domain restrictions, and graph labels.
38-48 min

Independent correction

Use the topic cards, drills, and worked solutions to improve one incomplete response. The corrected version should show the method, not just the answer.

48-55 min

Exit ticket

  1. Solve 2sin(x)=1 for 0≤x≤2π.
  2. State the domain and range of r(x)=-x+2+4.
  3. Write the first revision task you will complete at home and the evidence you will use to know it is done well.
