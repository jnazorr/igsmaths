Year 11 Mathematical Methods Unit 1

Revision Worksheet

Name: Class: Date:

Show clear working. Give exact answers unless a decimal approximation is requested. Label all graph features carefully.

Part A: Function Language

  1. For f(x)=3x-2+1, state the maximal domain, vertical asymptote, horizontal asymptote, and intercepts.
  2. Sketch the graph, labelling all key features.
  3. Write the function in full function notation.

Part B: Circular Functions

  1. Solve 2sin(x)-1=0 for 0≤x≤2π.
  2. Sketch y=2cos(3x)-1 for one full cycle.
  3. State the amplitude, period, maximum value, and minimum value.

Part C: Quadratics and Inverses

  1. Find the rule of a quadratic with turning point (-2,7) passing through (1,-2).
  2. For g:[4,∞)→R, g(x)=x-4+2, find g-1.
  3. State the domain and range of both functions.

Part D: Piecewise Modelling

  1. Create a piecewise function with exactly three pieces: one linear, one quadratic, and one circular function.
  2. Use adjacent domains that do not overlap.
  3. Sketch the full graph and label each endpoint clearly.

Part E: Probability

  1. A trial has success probability 0.4 and is repeated three times independently. Draw a tree diagram.
  2. Find the probability of exactly two successes.
  3. Find the probability of at least one success.
  4. Explain which calculation could be done using the complement.
